Git Common commands

  1. git init: initialize a new local git repo
  2. git add .: adds files to the staging area
  3. git commit -m "some message" : writes the changes to the files in staging
  4. git push -u origin main: pushes our changes to our remote repo

Terminal Commands

  • pwd: shows us the working directory
  • ls: shows us the contents within our working directory
  • cd: change directory
  • mkdir: create a folder
